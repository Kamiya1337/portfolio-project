# AGENTS.md

## Project

React + Vite + Tailwind portfolio website.

## Commands

- npm run dev
- npm run build
- npm run lint

## Rules

- Reuse existing components when possible.
- Do not introduce unnecessary dependencies.
- Remove dead code when discovered.
- Prefer small focused changes.
- Explain major refactors before implementing.

## UI

- Preserve existing content unless asked.
- Use Tailwind as primary styling system.
- Use Framer Motion for advanced animations.
- Maintain responsive layouts.

## Validation

Before finishing:

1. Run lint.
2. Run build.
3. Summarize changed files.
